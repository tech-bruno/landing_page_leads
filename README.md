💡 Sobre o projeto

Esta landing page foi desenvolvida com foco em **gerar leads qualificados**, utilizando uma estrutura estratégica de marketing digital e princípios de **UX Design** para maximizar conversões.

A página apresenta o método "Legado Construído", com abordagem persuasiva voltada para despertar interesse e incentivar o agendamento de uma sessão gratuita.

---

🧱 Estrutura da página

- Hero (chamada principal com CTA)
- Apresentação do método
- Benefícios estratégicos
- Depoimentos (prova social)
- Seção de credibilidade
- Formulário de captação de leads
- CTA final
- Footer institucional
